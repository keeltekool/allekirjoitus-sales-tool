# Professional Services

When your implementation requires custom integrations, bespoke form development, or specialized configuration beyond standard features, the professional services team handles the technical work - from initial architecture consultation through development, testing, and deployment.

- **What services are available** - custom API integrations connecting to your specific systems (HR platforms like Sympa, CRM systems, proprietary archiving solutions). E-form creation: translating your static PDFs into dynamic interactive web forms with validation logic, conditional fields, and CSS styling. PAdES cover page template creation with your branding and dynamic metadata fields. Document Templating setup: mapping JSON keys to PDF placeholder fields. Complex Active Directory/Entra ID integration with hierarchical department claim mapping. Custom domain setup including SSL certificate exchange and DNS configuration. Custom usage reporting generated via database queries

- **How engagements work** - before any work begins, the team assesses your requirements and presents a workload estimate. Simple e-forms typically take 1-2 days; complex integrations are scoped case by case. Once approved, work is billed on actual hours spent. All hourly work carries a minimum charge of 0.5 hours. Custom reports are billed as exactly 1 hour per report

- **Specialist consultation vs. development** - development and maintenance covers hands-on technical execution: coding, form building, API integration, workflow configuration, testing, and daily support requests. Specialist consultation covers strategic advisory: analyzing how to best digitalize your processes, defining technical architecture, setting 1-5 year targets, and managing complex multi-party implementation projects

- **What customers typically request** - the most common requests are custom e-form creation for specific business processes, API integration with internal systems, branded PAdES templates, SSO/AD configuration with complex group mappings, and custom reporting. Initial branding and portal customization is a standard one-time setup
